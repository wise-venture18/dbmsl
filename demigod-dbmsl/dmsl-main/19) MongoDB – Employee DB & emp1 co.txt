üóÑÔ∏è Code (with what it does)
use Employee;


Switches to (or creates) DB Employee.

db.createCollection("emp1");


Explicitly creates collection emp1 (optional‚ÄîMongo creates on first insert too).

db.emp1.insertMany([
 {eno:1, ename:"neha",  address:"pune",   sal:12000},
 {eno:2, ename:"nayan", address:"mumbai", sal:8000},
 {eno:3, ename:"ravi",  address:"pune",   sal:22000}
]);


Inserts 3 employee documents.

db.emp1.find();


Lists all documents.

db.emp1.find({ sal: { $gt: 5000 } });


Filter: salary greater than 5000.

db.emp1.find({ sal: { $lt: 15000 } });


Filter: salary less than 15000.

db.emp1.find({ sal: { $gt: 10000, $lt: 20000 } });


Range query (AND): 10k < sal < 20k.

db.emp1.updateMany(
  {},
  [ { $set: { sal: { $round: [ { $multiply: ["$sal", 1.10] }, 0 ] } } } ]
);


Pipeline update: gives 10% hike to every document.

$multiply increases salary by 1.10; $round rounds to 0 decimals.

After this: 12000‚Üí13200, 8000‚Üí8800, 22000‚Üí24200.

db.emp1.deleteMany({ sal: { $lt: 5000 } });


Deletes any records with salary < 5000 (none in this dataset; safe no-op).

db.emp1.renameCollection("employee1");


Renames collection emp1 ‚Üí employee1.

db.employee1.find({ ename: /^n/i });


Regex search: names starting with ‚Äòn‚Äô (case-insensitive).

Matches "neha", "nayan".

db.employee1.find().sort({ ename: 1 });


Sorts all docs by ename in ascending order (A‚ÜíZ).

use Employee1; 
db.dropDatabase();


Switches to DB Employee1 (just to show) and drops the entire database (dangerous; removes all collections inside).

use Employee; 
db.emp1.insertOne({ eno: 99 }); 
db.emp1.drop();


Back to Employee DB, recreates a new emp1 with one temp doc, then drops the collection emp1.
