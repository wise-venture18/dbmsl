-> PLSQL In Ubantu
 Step 1: Write Your PL/SQL Code in Notepad
You can use Notepad, VS Code, or Gedit to write your PL/SQL program.

Example: hello.sql

SET SERVEROUTPUT ON;
BEGIN
    DBMS_OUTPUT.PUT_LINE('Hello from PL/SQL on Ubuntu!');
END;
/

Save the file as:
/home/yourusername/hello.sql

Step 2: Connect to Oracle Using SQL*Plus
Open your terminal and connect to Oracle:
sqlplus system/password@localhost
(Replace system and password with your Oracle credentials.)

Step 3: Run Your PL/SQL File
Once connected, run:
@/home/yourusername/hello.sql

Output:
Hello from PL/SQL on Ubuntu!
